## 📌 How to Run (Locally)
1. Open `NUnit&MOQ_code.sln` in Visual Studio 2022
2. Restore NuGet packages:
   - `NUnit`
   - `NUnit3TestAdapter`
   - `Microsoft.NET.Test.Sdk`
   - `Moq`
3. Build the solution (`Ctrl+Shift+B`)
4. Open **Test Explorer** → Run All Tests
5. For SQL tasks, run scripts inside any SQL Server instance (e.g. SSMS)
